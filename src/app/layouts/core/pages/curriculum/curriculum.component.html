Web Development From the Ground Up

In this course we explore the basic fundamentals of web development and continue through the complexities of the web to reach our final destination and understand the complex fundamentals that every successfully web developer needs to live and breath. By the end of this course you will be able to create functional websites and web applications.

Languages/Skills Covered

  • HTML5
  • Bootstrap 4
  • Basic CSS

Climbing the Mountain of CSS

In this course we take our skills to the next level and learn how to build not just functionally pleasing web site and web applications but immaculate pieces of artwork. Our website and web applications are already naturally visually appealing, but what if we take what is already beautiful and raise the bar a few notches. By the end of this course you will understand complex styling concepts.

Languages/Skills Covered

  • Advanced CSS
  • CSS Preprocessor
  • CSS Frameworks

Decoding Javscript

In this course we will learn how to add some pop, motion and functionality to our websites and web applications. We will take our beautiful websites and web applications and add some real functionality to them.

Languages/Skills Covered

  • JavaScript
  • JQuery

Getting a better Angle with Angular

In this course we will learn everything there is to know about Angular. We will learn what as Web Development Framework is and why we use them. Angular is a platform that makes it easy to build applications within the web world. We will harness the power of angular to overcome the many different development challenged web developers face.

Languages/Skills Covered

  • Angular 7
  • Angular CLI

Getting mean with the MEAN stack

In this course we will take it its overdrive and learn the backend of the web development world. We will learn what an API is and how to build our very own. We will tie our frontend websites and web applications into our backend server to create a complete and finished product.

Languages/Skills Covered

  • MongoDB
  • Express
  • Angular 7
  • Node.js

Unveiling the Mysteries of TypeScript

In this course we will dive into typescript and why we should use it. As we travel through the course we will become proficient and comfortable using TypeScript. We will learn the pros and cons to using typeScript over JavaScript.

Languages/Skills Covered

  • TypeScript
